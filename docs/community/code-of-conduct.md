# Code of Conduct

All participants in the GlowBack community are expected to abide by our
Code of Conduct. The full text lives in
[CODE_OF_CONDUCT.md](https://github.com/LatencyTDH/GlowBack/blob/main/CODE_OF_CONDUCT.md)
at the repository root.

## Summary

- Be welcoming and inclusive.
- Be respectful of differing viewpoints.
- Accept constructive criticism gracefully.
- Focus on what is best for the community.
- Show empathy towards others.

Unacceptable behavior includes harassment, trolling, and publishing private
information without consent.

Report violations by opening an issue or contacting the maintainers directly.
