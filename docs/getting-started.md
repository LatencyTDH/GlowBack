# Getting Started

## Prerequisites

- Rust (latest stable)
- Python 3.8+ (for Python bindings)

## Clone and Test

```bash
git clone https://github.com/LatencyTDH/GlowBack.git
cd GlowBack
cargo test --workspace
```

## Run an Example

```bash
cargo run --example basic_usage -p gb-types
```

## Launch the UI

```bash
cd ui
python setup.py
# Opens http://localhost:8501
```
